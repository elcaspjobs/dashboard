# Social Media Dashboard Analytics System

Асинхронная система сбора, агрегации и анализа статистики из социальных сетей (YouTube, LinkedIn, Twitter, Facebook, Instagram).  
Проект реализует ETL-подобную обработку данных: получение статистики через внешние API, сохранение в базу данных и предоставление агрегированных отчётов.

## Архитектура

Система построена по принципам **чистой архитектуры**:
- **Domain-Driven Design (DDD)** — чёткое разделение слоёв: домен, приложение, инфраструктура.
- **Dependency Inversion** — интеракторы зависят от абстракций (`UnitOfWork`, `HttpService`).
- **Асинхронность (asyncio)** — высокая производительность при работе с I/O-операциями.

### Слои
- **Domain** — сущности (`Dashboard`, `Publication`, `Connection`), бизнес-правила.
- **Application** — интеракторы, DTO, порты взаимодействия.
- **Infrastructure** — реализация репозиториев, HTTP-клиент, поллер задач, интеграции с соцсетями.

## Функционал

- Автоматический опрос публикаций за заданный период.
- Асинхронный сбор статистики (лайки, просмотры, комментарии, репосты) через внешнее API.
- Поддержка нескольких соцсетей через фабрику `SocialFactory`.
- Устойчивость к ошибкам: retry-логика, семафоры, изоляция сбоев.
- Агрегация данных в реальном времени.
- Гибкая система логирования (structlog).
